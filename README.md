# Automated Report Generator

This project reads data from `data.csv`, analyzes it, and generates a formatted PDF report using Python and FPDF.

## Features
- Reads data from CSV
- Calculates average, highest and lowest marks
- Generates a clean PDF report
- Uses FPDF library

## How to Run

Install dependencies:
pip install pandas fpdf

Run the project:
python report_generator.py

A PDF file named **sample_report.pdf** will be created automatically.
